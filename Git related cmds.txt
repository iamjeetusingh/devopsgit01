1. create GitHub account
2. create a public repo
3. install git on your machine
4. create a new folder and open git bash inside it.
5. initialize the git (git init)
6. change the branch (git branch -M main)
7. configure the username & email id 
	(git config --global user.name "your-name")
	(git config --global user.email "your@email.com")
8. create some files (touch)
9. check the status (git status)
10. add these file to staging area (git add .)
11. add a commit message (git commit -m "add your message")
12. add the GitHub/remote repo (git remote add origin https://github.com/jitendrastomar5593/devopsgit01.git)
13. type username & password OR token OR SSH-keys (if for the 1st time).
14. push the data to remote repo (git push -u origin main)
15. go to GitHub repo & refresh the web page.
